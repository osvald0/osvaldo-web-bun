# Osvaldo personal website

This is my personal website project.

## Getting Started

### Install Bun

```
curl -fsSL https://bun.sh/install | bash
```

### Clone the repo

```
git@github.com:osvald0/osvaldo-web-bun.git
```

### Development

```
bun dev
```

Open http://localhost:3000 with your browser to see the result.

### Production

Build
```
bun run build
```

Run
```
bun start
```
